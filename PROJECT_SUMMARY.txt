KEY FEATURES IMPLEMENTED
===============================================================================

DASHBOARD (src/app/session/[id]/page.tsx):

1. PHOTO SELECTION
   - Click to select/deselect images
   - Numbered badges show selection order
   - Drag-reorder with arrow buttons
   - Grid layout with visual feedback

2. CAPTION BUILDER
   - Three sections: Openers, Bodies, Closers
   - Pre-written options OR custom text toggle
   - Hashtag chips (clickable toggle)
   - Live caption composition

3. SCHEDULING
   - "Nu direct" - immediate
   - "Eerstvolgende standaardtijd" - next standard time
   - "Vandaag standaardtijd" - today at standard time
   - "Morgen standaardtijd" - tomorrow at standard time
   - "Kies datum en tijd" - custom date/time
   - Standard times: Weekdays 19:00, Weekends 13:00 (Amsterdam time)

4. PLATFORM SELECTION
   - Instagram toggle
   - Facebook toggle
   - Both selected by default

5. INSTAGRAM PREVIEW
   - Phone mockup (320px width)
   - Shows first selected image
   - Live caption preview
   - Updates as you compose

6. ADDITIONAL FEATURES
   - Word counter
   - Form validation
   - Error handling
   - Loading states
   - All text in Dutch

API ENDPOINTS:

1. POST /api/sessions
   Creates session, stores in Supabase, returns session_id and URL

2. GET /api/sessions
   Lists all pending sessions

3. POST /api/schedule
   Complete workflow:
   - Download images from Shopify CDN
   - Upload to Zernio
   - Create post on Zernio
   - Store post record in Supabase
   - Update session status
   Returns: post_id, zernio_post_id

===============================================================================
DATABASE SCHEMA
===============================================================================

sessions table:
  id                UUID primary key
  product_ids       TEXT[]
  images            JSONB
  caption_options   JSONB
  status            TEXT ('pending', 'scheduled', 'published')
  created_at        TIMESTAMP
  updated_at        TIMESTAMP

posts table:
  id                UUID primary key
  session_id        UUID (foreign key → sessions)
  caption           TEXT
  image_urls        TEXT[]
  platforms         TEXT[] (instagram, facebook)
  scheduled_for     TIMESTAMP
  timezone          TEXT
  zernio_post_id    TEXT
  status            TEXT ('scheduled', 'published', 'failed')
  created_at        TIMESTAMP
  updated_at        TIMESTAMP

QUICK START
===============================================================================

1. Install dependencies:
   cd /sessions/bold-funny-sagan/woodykids-poster-app
   npm install

2. Set up environment:
   cp .env.example .env.local
   # Fill in Supabase and Zernio credentials

3. Create database tables (see README.md for SQL)

4. Run development server:
   npm run dev
   # Open http://localhost:3000

5. Test the flow:
   - Home page shows pending sessions (or "no sessions" message)
   - Create a session via /api/sessions endpoint
   - Open session from home page
   - Build post (select photos, compose caption, select hashtags)
   - Choose platform(s) and schedule time
   - Submit to /api/schedule
